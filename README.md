# 🩺 Health Risk Analysis Web App

## 📌 Overview
This project is a full-stack machine learning demo that estimates health risk levels based on key health indicators.

It is built to demonstrate a Flask API, a responsive frontend, and a basic scikit-learn prediction workflow.

---

## 🚀 Features
- Predicts health risk (Low / Moderate / High)
- Displays risk score (%) and interpretation
- Responsive web interface (mobile + desktop)
- Input validation for real-world usage
- Clean API structure using Flask

---

## 🧠 Machine Learning
- Model: Logistic Regression
- Preprocessing: StandardScaler
- Training data: synthetic demo data generated by `train_model.py`
- Features used:
  - Age
  - BMI
  - Blood Pressure
  - Blood Sugar
  - Cholesterol
  - Activity Level
  - Smoking Status
  - Family History

> **Important:** This app is a portfolio/demo screening tool, not a medical device or clinical diagnosis system. The included model is trained on synthetic data and should be replaced with validated real-world data before any real screening or health decision use.

---

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- ML: Scikit-learn
- Data: Pandas, NumPy

---

## 📊 Use Case
This system can be used for:
- Portfolio demonstration

- Flask API and frontend integration practice

## ▶️ How to Run Locally

```bash
git clone https://github.com/princessroffy/health-risk-analysis-app.git
cd health-risk-analysis-app
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python train_model.py
python app.py
```

Open the app at `http://127.0.0.1:5000`.

To enable Flask debug mode for local development:

```bash
FLASK_DEBUG=1 python app.py
```
