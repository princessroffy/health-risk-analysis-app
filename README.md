# 🩺 Health Risk Analysis App

##  Overview

The Health Risk Analysis App is an NGO-ready digital health screening tool designed to assess individual health risk levels and generate structured reports for community health programs.

This project demonstrates how **AI, data analysis, and simple machine learning models** can support **early health risk detection and decision-making** in underserved communities.

---

## ❗ Problem

Many communities lack access to early health screening and structured health data systems.

NGOs and health outreach programs often:
- Rely on manual assessments
- Lack proper data tracking
- Struggle to identify high-risk individuals early
- Cannot generate structured reports for stakeholders

---

##  Solution

This application provides:
- Digital health risk assessment
- Instant classification of risk levels
- Automated report generation
- Data storage for analysis and tracking

---

##  Features

- 🧾 Input patient/beneficiary health data  
- 🧠 Machine Learning-based risk prediction  
- 📊 Risk classification (Low / Medium / High)  
- 📄 PDF report generation for each individual  
- 📈 Data tracking for community health programs  
- 📥 Downloadable results

##  Model Explanation

The model uses a supervised machine learning approach to classify individuals based on input health indicators.

It considers:
- Age
- BMI
- Glucose levels
- Blood pressure
- Other health factors

The model outputs a probability-based classification:
- Low Risk
- Medium Risk
- High Risk
